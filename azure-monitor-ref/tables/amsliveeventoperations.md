# AMSLiveEventOperations

 Contains logs related to a Live Event. Logs are sent when an encoder connects, disconnects, or if there is a discontinuity in the media data.

## Categories

- Audit
- Azure Resources
## Solutions

- LogManagement
## Resource types

- Media Services
