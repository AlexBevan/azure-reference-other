# AHDSDicomAuditLogs

 Data plane audit logs of privileged actions made against Azure Health Data DICOM service. For example, storing a DICOM instance.

## Categories

- Audit
- Azure Resources
## Solutions

- LogManagement
## Resource types

- Health Data Services
