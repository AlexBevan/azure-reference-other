# ADReplicationResult

 The AD Replication Status solution regularly monitors your Active Directory environment for any replication failures.

## Categories

- Workloads
## Solutions

- AD Replication Status
## Resource types

- Virtual machines
- VMware
- Azure Stack HCI
- System Center Virtual Machine Manager
