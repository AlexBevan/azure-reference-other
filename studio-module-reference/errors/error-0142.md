# Error 0142  
 Exception occurs when the system cannot load certificate to authenticate.  
  
## Resolution  
  
|Exception Messages|  
|------------------------|  
|The certificate cannot be loaded.|  
|The certificate {0} cannot be loaded. Its thumbprint is {1}.|
