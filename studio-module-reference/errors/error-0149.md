# Error 0149
**Error 0149**  
  
 The user file does not exist inside GitHub bundle.  
  
 This error in Azure Machine Learning occurs when the specified file cannot be found.  
  
## Resolution  
  
|Exception Messages|  
|------------------------|  
|GitHub file is not found.|  
|GitHub file is not found.: {0}|
