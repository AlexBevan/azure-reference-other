# Error 0051  
 Exception occurs in the case when several output files are the same.  
  
### Resolution  
 This error in Azure Machine Learning is unused and will be deprecated.  
  
|Exception Messages|  
|------------------------|  
|Specified files for outputs cannot be the same.|
