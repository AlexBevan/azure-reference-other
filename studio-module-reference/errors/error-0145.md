# Error 0145
**Error 0145**  
  
 Cannot create the replication directory for some reason.  
  
 This error in Azure Machine Learning occurs when the module fails to create the specified directory.  
  
## Resolution  
  
|Exception Messages|  
|------------------------|  
|Cannot create replication directory.|
