# Error 0050  
 Exception occurs in the case when input and output files are the same.  
  
### Resolution  
 This error in Azure Machine Learning is unused and will be deprecated.  
  
|Exception Messages|  
|------------------------|  
|Specified files for input and output cannot be the same.|
